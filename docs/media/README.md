# Capturas y demos

Coloca aquí el material visual del README. Lo que necesitas grabar en tu máquina:

## Prioridad 1 — el GIF que vende (`demo-fusion.gif`)
15-20 segundos mostrando:
1. Escribes una pregunta que active varias IAs (ej: "compara microservicios vs monolito y dame ejemplos de código").
2. Se ven las 2-3 IAs respondiendo en paralelo (los badges de colores).
3. Aparece "✨ Respuesta unificada" y la IA de salida la pule.

Herramienta: ScreenToGif (Windows) o Kap (Mac). Exporta a <5MB, 720p.

## Prioridad 2 — capturas estáticas (PNG, 1280px ancho)
- `shot-chat.png` — una conversación con respuesta multi-IA y markdown bien renderizado (código con colores).
- `shot-models.png` — el panel de modelos con varias IAs activas.
- `shot-mlops.png` — la pantalla de métricas/MLOps con datos.

## Consejo
Usa el tema oscuro (por defecto), ventana limpia, y una pregunta que muestre
código + tabla + texto para que se vea el markdown completo.
